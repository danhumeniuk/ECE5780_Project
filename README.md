If you decide to use the Gyroscope and USART, the pinlayout is as follows:

Gyroscope:  PB13 connected to 10k resistor and 3V
            PB15 and PB11 Connected to the same 10k resistor and 3V
           
USART:      PA9  RX
            PA10 TX
